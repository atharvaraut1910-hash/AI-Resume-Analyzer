# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python, Django, SQLite, Bootstrap, and Google Gemini AI**. This application helps job seekers evaluate their resumes by calculating ATS scores, matching resumes with job descriptions, generating AI-based resume analysis, and creating professional cover letters.

---

## 🚀 Features

- 📄 Upload Resume (PDF/DOCX)
- 📑 Resume Text Extraction
- 🎯 ATS Score Calculation
- 📋 Job Description Match
- 🤖 AI Resume Analysis using Google Gemini
- ✍️ AI Cover Letter Generation
- 📜 Resume History
- 📥 PDF Report Download
- 🔐 Login System
- 📱 Responsive Bootstrap UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/atharvaraut1910-hash/AI-Resume-Analyzer.git
```

### Go to Project Folder

```bash
cd AI-Resume-Analyzer
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
